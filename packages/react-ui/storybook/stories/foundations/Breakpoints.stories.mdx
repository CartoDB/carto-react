# Breakpoints

These are the keys available:

```js
xs: 320;
sm: 600;
md: 960;
lg: 1280;
xl: 1600;
```

Examples of use and its result:

```css
breakpoints.between('xs', 'sm') -> @media (min-width:320px) and (max-width:599.95px)
breakpoints.between('sm', 'md') -> @media (min-width:600px) and (max-width:959.95px)
breakpoints.between('md', 'lg') -> @media (min-width:960px) and (max-width:1279.95px)
breakpoints.between('lg', 'xl') -> @media (min-width:1280px) and (max-width:1599.95px)

breakpoints.only('xs') -> @media (min-width:320px) and (max-width:599.95px)
breakpoints.up('xs') -> @media (min-width:320px)
breakpoints.down('xs') -> @media (max-width:319.95px)

breakpoints.not('xs') -> @media (min-width:600px)
breakpoints.not('sm') -> @media not all and (min-width:600px) and (max-width:959.95px)
```
